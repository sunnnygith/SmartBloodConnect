# 🩸 Smart Blood Connect – LPU

A Java web application for the LPU campus community to connect blood donors and recipients.

## Tech Stack
- **Backend:** Java (JSP/Servlet)
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Tools:** VS Code, Apache Tomcat, iText PDF

## Features
- Hostel-wise donor search
- Blood request system (pending → approved)
- PDF certificate generation
- Admin dashboard
- Campus Hospital awareness events
